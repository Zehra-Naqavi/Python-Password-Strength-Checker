# Python-Password-Strength-Checker
A command-line tool that checks how strong a password is.

**What it does**
- Checks length (8+ and 12+ characters)
- Checks for uppercase and lowercase letters
- Checks for numbers and symbols
- Gives a score out of 6 and a strength rating
- Suggests improvements if the password is weak

**Why I built it**
I wanted to learn Python and was interested in cybersecurity basics. 
This project taught me how to use functions, loops, and regular 
expressions (re module) to analyse text.

**What I learned**
- Python functions and return values
- Regular expressions with the re module
- While loops and user input
- Breaking a problem into small checks

**How to run it**
1. Make sure Python is installed on your computer
2. Download or clone this repo
3. Open terminal and run: python password_checker.py
4. Type a password and see your results!
